- "[라이브러리/패턴] 함정 N가지 + 회피 + 우리 토이 적용성"
- "내가 의식 못 하고 있을 함정 알려줘"

- "한 단계 끝나면 commit / 검증 가능한 단위로"

## 의문 풀이
